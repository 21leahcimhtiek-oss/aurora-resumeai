# ResumeAI Pro

> AI resume builder and optimizer

## Features
- ATS optimization
- AI rewriting
- Cover letters
- LinkedIn sync
- Job match scoring

## Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- OpenAI GPT-4o-mini
- Stripe Subscriptions
- Vercel deployment

## Quick Start

```bash
npm install
cp .env.example .env.local
# Fill in API keys
npm run dev
```
